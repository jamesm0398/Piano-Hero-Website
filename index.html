What is Piano Hero?

Piano Hero is a SET capstone project that will allow a user to learn the piano by using their own keyboard, and a strip of LED lights that blink in sequence according to what notes should be played for a particular song. Users can choose a wav file and have it converted to a MIDI file using our online service. Piano Hero is currently under development and will be released April 23rd 2020. See our Semester Goals page for more details.

Components

WAV to MIDI File Converter

will be placed on top of the user's piano keyboard

Microcontroller

LED Strip

Desktop Application

Features Include:

Ability to select a WAV file and convert it to a MIDI file using our cloud service

Wirelessly download MIDI file to the microcontroller

Select a speed at which to play the song from the desktop app

Setup Guide

1. Ensure you have the latest version of the Piano Hero Desktop App installed

2. Connect microcontroller to the same network as your pc

3. Place LED strip in precisely marked position on your keyboard
